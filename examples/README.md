# Example

A small, self-contained example of what `img2dots` produces.

| File | What it is |
| --- | --- |
| `square.png` | The input image (512×512, RGBA) — a rounded square with a blue-to-orange diagonal gradient on a transparent background. |
| `square.md` | The rendered output, produced by the command below. Open it in a LaTeX-capable Markdown viewer (MathJax / Obsidian) to see the dot image. |
| `make_example.py` | The script that generates `square.png`, so the example is fully reproducible. |

## How the example was generated

```
python examples/make_example.py                                   # creates square.png
img2dots examples/square.png -o examples/square.md --max-size 40  # renders square.md
```

`square.md` is rendered with default options only — the transparent margin and rounded corners are
skipped automatically (default `--alpha-threshold` of 50%), so the square sits on an empty canvas.

> test (`tests/test_examples.py`) ever fails after a Pillow upgrade.
